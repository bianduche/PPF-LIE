## 1. 光子分布先验

### 1.1 物理背景

在现代数码相机成像的底层物理过程中，图像传感器（CMOS/CCD）通过收集光子并将其转换为电子来形成图像。

### 1.2 光照条件的影响

| 条件 | 光子数量 | 噪声分布 | 可近似为 |
|------|----------|----------|----------|
| 良好光照 | 大量光子 | 严格泊松 | 高斯分布 |
| 极端低光 | 稀疏光子 | 严格泊松 | **不可近似** |

### 1.3 泊松分布建模

在极暗环境中，像素x处的离散光子计数N(x)严格服从以连续辐射场λ(x)为期望的**泊松分布**：

$$N(x) \sim \text{Poisson}(\lambda(x)) = \frac{\lambda(x)^{N(x)} e^{-\lambda(x)}}{N(x)!}$$

### 1.4 简化假设

为保持扩散框架的闭式可处理性，本文将成像模型抽象为纯泊松过程。在此极端光子匮乏条件下，信号依赖的泊松散粒噪声主导退化空间。

## 2. 光子概率场建模

### 2.1 核心任务

从离散观测N(x)设计深度神经网络，以参数化方式估计底层连续光子概率场λ(x)。

### 2.2 传统方法的局限

传统深度学习将网络训练为确定性映射函数：
$$f_\theta: I_{in} \rightarrow I_{clean}$$

在极稀疏泊松空间中，由于观测N(x)存在大量零值和孤立离散峰，确定性网络极易陷入局部最优，强行拟合离散噪声点。

### 2.3 概率场参数估计

设神经网络$f_\theta$作为概率场的参数估计器，其输出$\lambda_0 = f_\theta(N(x), \Phi)$表示网络预测的连续期望场，其中Φ表示从原始暗输入提取的多尺度条件特征。

### 2.4 最大似然估计

根据最大似然估计原理，给定离散观测N(x)找到最优连续场λ(x)等效于最大化联合泊松似然：

$$\theta^* = \arg\max_\theta \prod_{x \in \Omega} \frac{\lambda_0(x)^{N(x)} e^{-\lambda_0(x)}}{N(x)!}$$

取负对数并忽略常数项，参数估计的数学本质是最小化泊松NLL：

$$\mathcal{L}_{MLE}(\theta) = \sum_{x \in \Omega} [\lambda_0(x) - N(x)\log\lambda_0(x) + \delta]$$

其中δ是防止数值不稳定的小常数。

## 3. 离散扩散过程

### 3.1 连续 vs 离散扩散

| 特性 | 连续扩散（传统） | 离散扩散（PPF-LIE） |
|------|-----------------|---------------------|
| 噪声类型 | 高斯噪声 | 泊松噪声 |
| 状态空间 | 连续实数值 | 离散整数计数 |
| 时间步态 | 连续缩放 | 加性马尔可夫链 |
| 物理一致性 | 违反光子统计 | 完全符合 |

### 3.2 离散前向过程

不同于连续高斯缩放，我们将前向扩散过程建模为**离散马尔可夫链**，通过注入随机泊松散射噪声逐步破坏光子状态：

$$q(N_t|N_{t-1}) = \text{Poisson}(N_t - N_{t-1}; \beta_t)$$

其中$\beta_t > 0$表示方差调度，控制时间步t注入的噪声光子的期望速率。

状态转移可表示为：
$$N_t = N_{t-1} + \epsilon_t, \quad \epsilon_t \sim \text{Poisson}(\beta_t)$$

### 3.3 物理约束

此加性泊松前向过程**严格强制**光子计数的非负性：
$$N_t \geq N_{t-1}, \quad \forall t$$

这完全符合物理现实：前向腐败过程中只能添加噪声光子。

### 3.4 方差调度

采用$\beta_t$的线性方差调度，值在T=1000时间步内从$10^{-4}$到$2 \times 10^{-2}$，确保累积光子噪声$\beta_T = \sum_{i=1}^T \beta_i$在数值稳定范围内而不会爆炸。

### 3.5 闭式边缘化分布

利用泊松分布的**无限可分性**和**可列可加性**，可数学推导闭式边缘分布$q(N_t|N_0)$。由于独立泊松分布随机变量之和本身仍服从泊松分布：

$$q(N_t|N_0) = \text{Poisson}(N_t - N_0; \beta_t)$$

其中累积光子噪声参数$\bar{\beta}_t = \sum_{i=1}^t \beta_i$。

### 3.6 贝叶斯后验

给定真实初始状态$N_0$，后验转移概率$q(N_{t-1}|N_t, N_0)$可使用贝叶斯定理精确计算：

$$q(N_{t-1}|N_t, N_0) = \frac{q(N_t|N_{t-1}) q(N_{t-1}|N_0)}{q(N_t|N_0)}$$

### 3.7 网络参数化

**核心创新**: 网络参数化为直接预测底层连续期望光子率：
$$\lambda_0 = f_\theta(N_t, t)$$

通过预测λ_0，弥合了离散观测与连续物理流形之间的差距。

### 3.8 训练目标

绕过复杂的离散KL散度，采用时间加权泊松负对数似然损失函数：

$$\mathcal{L}_{diffusion} = \mathbb{E}_{t,N_0,\epsilon}[\lambda_0(x) - N_0(x)\log\lambda_0(x) + \delta]$$

## 4. 辐射场重建与颜色一致性

### 4.1 问题

虽然PPF扩散模型优雅地恢复了连续光子概率场$\lambda_0 \in \mathbb{R}^{H \times W \times 3}$，但现代下游视觉任务在标准sRGB色彩空间运行。

### 4.2 CCM模块

引入辐射重建和颜色一致性模块（CCM），结合恢复的连续场$\lambda_0$和原始低光输入提取的条件特征$\Phi_{I_{in}}$：

$$I_{out}(x) = \mathcal{M}_{color}(\lambda_0(x), \Phi_{I_{in}}(x))$$

其中$\mathcal{M}_{color}$表示可学习的颜色校正映射，Φ表示条件编码器。

### 4.3 CCM架构

CCM预测局部颜色仿射变换来生成最终增强图像：

```python
# 伪代码
for each pixel location:
    extract local features from lambda_0
    extract conditional features from Phi
    predict affine transformation (scale, bias)
    apply transformation
```

## 5. 总体优化目标

### 5.1 总损失函数

$$\mathcal{L}_{total} = \mathcal{L}_{diffusion} + \gamma_1 \mathcal{L}_{percep} + \gamma_2 \mathcal{L}_{color} + \gamma_3 \mathcal{L}_{tv}$$

### 5.2 各损失项

| 损失项 | 符号 | 作用 |
|--------|------|------|
| 扩散损失 | $\mathcal{L}_{diffusion}$ | 泊松NLL，优化光子场恢复 |
| 感知损失 | $\mathcal{L}_{percep}$ | 深度特征对齐 |
| 颜色损失 | $\mathcal{L}_{color}$ | 显式监督颜色保真度 |
| TV损失 | $\mathcal{L}_{tv}$ | 强制空间平滑 |

### 5.3 感知损失

$$\mathcal{L}_{percep} = \|\phi(I_{pred}) - \phi(I_{gt})\|_1$$

使用预训练VGG网络的深层特征。

### 5.4 损失权重

| 数据集 | $\gamma_1$ | $\gamma_2$ | $\gamma_3$ |
|--------|-----------|-----------|-----------|
| LOL | 0.1 | 0.05 | 0.01 |
| ML | 0.15 | 0.08 | 0.02 |
